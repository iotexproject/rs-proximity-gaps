P3 structural sweep: K(f) for 3-pos sparse f at deployment-closure scales

Method:
  - Enumerate all 3-element DFT supports {a, b, c} subset Z/n_0
  - Filter to far-from-RS_{k_0} (at least one position >= k_0)
  - Compute K_struct = sum over arising pairs of orbit sizes
  - Verify K_struct <= n_0/4 + 2 (Theorem 5.1)

RESULT: PASS at all three deployment-closure scales.

This validates Theorem 5.1's bound n_0/4 + 2 at the support level
for every three-position-sparse f far from RS_{k_0}, independent
of the q-dependent arising-pair sweep that underlies
Corollary 5.2.  The bound is achieved when the support's
largest arising-pair gap equals k_0 / 2, contributing orbit size
n_0 / (k_0 / 2) = 8 (for n_0 = 32, k_0 = 8).

Open Problem P3 (#381) is the EXTENSION of this bound to
non-3-pos-sparse f, which requires either:
  (a) a moment-bound reduction (paper 1 c >= 2 framework on
      branch feat/c2-moment-bound), or
  (b) a Sudan-level list-decoder + worst-case empirical sweep
      (out of scope here; would handle f with sparsity 4+ at
       small scales, but does not scale to ABF (n_0, k_0) =
       (2^{21}, 2^{20})).
