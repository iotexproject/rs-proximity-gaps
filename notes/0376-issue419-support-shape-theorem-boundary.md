## Current reduced target

After Notes 0356--0360, the remaining nonzero-alpha primitive branch is:

```text
alpha1 != 0,
rank(W)=2,
no-full saturated component S,
W has no nontrivial dyadic row-span stabilizer.
```

The desired contradiction is now equivalent to the following support-shape
statement.

> **Support-shape theorem.**  In the quotient-`C4` defect-allocation system,
> after saturating by `alpha1 != 0` and `rank(W)=2`, any nonzero-alpha
> solution has the two residual quotient rows supported on opposite parity
> classes.

Opposite parity support gives a half-turn row-span stabilizer, which is
already charged by Notes 0347--0351.  Thus this theorem would close the
remaining primitive nonzero-alpha branch.

---

## What is already an algebraic identity

Write the folded residual rows as

```text
u_alpha(r) = c_{4r}   + alpha1 c_{4r+1},
v_alpha(r) = c_{4r+2} + alpha1 c_{4r+3}.
```

Two collapse mechanisms are immediate.

### Same-folded cancellation

If a row contains both entries in one folded fiber and

```text
c_{4r} + alpha1 c_{4r+1}=0
```

or

```text
c_{4r+2} + alpha1 c_{4r+3}=0,
```

then the support of that row loses the corresponding quotient monomial.  In
all observed nonzero-alpha shapes this either leaves one residual row zero or
leaves the two rows proportional, hence lies outside the `rank(W)=2`
saturation.

The catalog-wide Groebner certificate in Note 0360 verifies this implication
for every observed same-folded shape.

### Same-parity proportionality

If both residual rows survive on the same single parity class, then the
rank-2 minors are exactly the obstruction.  In the observed rank-or-parity
degenerate shapes, saturation by the rank-2 minors returns the unit ideal.

This is also certified for every observed shape by Note 0360.

## What remains unproved

The missing all-scale step is not one of the two collapse mechanisms above.
It is the exclusion of a mixed-parity rank-2 shape:

```text
alpha1 != 0,
rank(W)=2,
no same-folded cancellation,
at least one of u_alpha or v_alpha has both parities,
and W has no half-turn stabilizer.
```

The base panels show no such shape:

```text
q=97, four-support:    15 unique shapes, 0 mixed-parity rank-2
q=193, five-support:   24 unique shapes, 0 mixed-parity rank-2
q=1153, four-support:   2 unique shapes, 0 mixed-parity rank-2
```

The only observed mixed-parity rows occur after same-folded cancellation has
already killed the other row, so they are rank-collapse shapes.

---

## Candidate proof route

The quotient-`C4` dual system gives, for each no-full component `S`,

```text
H_S u_alpha = 0,     H_S v_alpha = 0,
```

where `H_S` is the high-tail parity-check matrix for `RS_k(S)`.
Equivalently:

```text
H_S u_0 + alpha1 H_S u_1 = 0,
H_S v_0 + alpha1 H_S v_1 = 0.
```

The all-scale proof should show that, after inverting:

1. `alpha1`;
2. same-folded cancellation factors; and
3. at least one rank-2 minor of `W`,

the ideal generated by these two affine systems implies the half-turn parity
eigenspace equations:

```text
u_even u_odd = 0,
v_even v_odd = 0,
u_even v_even = 0,
u_odd  v_odd  = 0.
```

This is the precise algebraic formulation of the support-shape theorem.

## Why more base enumeration is low ROI

The q=97/q=193/q=1153 catalogs already show that the nonzero-alpha locus is
confined to rank-collapse and half-turn shapes in every complete panel where
it appears.  Random larger-support stress in Note 0353 found only the
`alpha1=0` degeneration.

More enumeration can catch bugs, but it will not itself supply the missing
all-scale implication.  The useful next move is to construct a symbolic
template for the forbidden mixed-parity rank-2 shape and prove its saturated
ideal is empty before field specialization.

## Next concrete artifact

Build a shape-template verifier that does not start from observed examples.
It should enumerate abstract mixed-parity support templates with:

```text
no same-folded cancellation,
rank2 minor inverted,
alpha1 inverted,
one row mixed parity or both rows same parity,
```

and ask whether any no-full tail matrix `H_S` can satisfy the affine system.

If every such template saturates to the unit ideal, the resulting Groebner
identities are candidates for a hand proof of the support-shape theorem.  If a
template survives, it is the first precise obstruction to #419.
