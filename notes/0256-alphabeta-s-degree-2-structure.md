## The fact

In the Kummer ring A = F(s)[z]/(z^h - s), the master form αβ satisfies:

```
[z^c] (αβ)  has s-degree ≤ 2,    for all c = 1, ..., h-1.
```

Specifically, [z^c] (αβ) ∈ span{1, s, s²} as a polynomial in s.

(The [z^0] coefficient has support {s^0, s^1, s^2, s^3, s^8} due to the
P_0 R_0 = s⁸ - 1/16 contribution.)

## Algebraic derivation

Recall (Note 0252) the master identity:

```
αβ = (s⁸ - 1/16) + σ X + δ U + (X² - U²)
```

where:
- σ = P_0 + R_0 = 2s⁴ + s² - s   (degree 4 in s)
- δ = R_0 - P_0 = -2s³ + s² + 1/2   (degree 3 in s)
- X = Σ x_c z^c   (no s)
- U = Σ u_c z^c, u_c = (x_c - W_c)/2 + x_c · s   (degree 1 in s)

### s⁴ cancellation

[z^c] (σ X) = σ · x_c, with s⁴ part = 2 s⁴ x_c.
[z^c] (δ U) has s⁴ part = -2 s³ · x_c · s = -2 s⁴ x_c.
[z^c] (X² - U²) has s ≤ 2.
Wraparound contributions (× s): s · [z^{c+h}] X = 0; s · [z^{c+h}] U = 0;
contribute up to s³.

Total s⁴ at [z^c] αβ in A:  2 x_c + (-2 x_c) + 0 + 0 = **0**.

### s³ cancellation

[z^c] (σ X) has s³ part = 0 (σ has no s³).
[z^c] (δ U) has s³ part = W_c (computed: -s³(x_c-W_c) + s³ x_c = s³ W_c).
Wraparound: s · ([z^{c+h}] s² part of -U²) = s · (-W_c) = -s³ W_c.

Total s³ at [z^c] αβ in A:  W_c + (-W_c) + 0 = **0**.

### Higher s degrees (s⁵..s⁷)

All s-degrees > 2 from direct contributions cancel against wraparound
contributions (× s shifts each by one). The analysis is symmetric to
s⁴ and s³ above, with all coefficients telescoping to 0.
