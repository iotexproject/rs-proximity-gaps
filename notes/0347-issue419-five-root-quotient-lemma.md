## Statement

Let `F` have odd characteristic and contain `mu_8`.  Put

```text
V = span_F {1, y, y^4, y^5}  subset F[y]/(y^8-1).
```

Let

```text
mu_8^+ = {y in mu_8 : y^4= 1},
mu_8^- = {y in mu_8 : y^4=-1}.
```

For `T subset mu_8` with `|T|=5`, the evaluation map

```text
ev_T : V -> F^T
```

has nonzero kernel iff

```text
T = mu_8^+ union {t},  t in mu_8^-,
```

or

```text
T = mu_8^- union {t},  t in mu_8^+.
```

In either case the kernel is one-dimensional.  If `t` is the singleton point
outside the full parity coset, then a generator is

```text
f_{s,t}(y) = (y^4-s)(y-t),      s in {+1,-1},  t^4=-s.
```

The high direction of this generator in the basis

```text
1, y, y^4, y^5
```

has ratio

```text
[y^5]f / [y^4]f = -t^{-1}.
```

---

## Proof

Write any element of `V` as

```text
f(y)=a+b y+c y^4+d y^5.
```

On the parity coset `mu_8^s = {y : y^4=s}`, this restricts to a linear
polynomial in `y`:

```text
f(y) = (a+s c) + (b+s d)y.
```

A nonzero linear polynomial has at most one root.  Therefore, if `f` vanishes
at two or more points in the same parity coset, its restriction to that coset
is identically zero:

```text
a+s c = 0,       b+s d = 0.
```

Now let `|T|=5`.  Since the two parity cosets each have size four, one of
them contains at least three points of `T`.

If `T` meets both parity cosets in at least two points, then the restriction of
`f` is identically zero on both cosets.  The four equations

```text
a+c=0,  b+d=0,  a-c=0,  b-d=0
```

force `a=b=c=d=0` because `char(F)` is odd.  Hence no nonzero kernel exists.

The only remaining possibility for a nonzero kernel is a `(4,1)` split across
the two parity cosets.  Suppose

```text
T = mu_8^s union {t},      t^4=-s.
```

Vanishing on the full coset `mu_8^s` imposes

```text
a+s c=0,       b+s d=0.
```

This leaves the two-dimensional family

```text
f(y) = (y^4-s)(c+d y).
```

Vanishing at the remaining point `t` gives

```text
(t^4-s)(c+d t) = -2s(c+d t)=0.
```

Since `char(F)` is odd, this is the single nonzero equation

```text
c+d t=0.
```

Thus the kernel is one-dimensional and generated by

```text
(y^4-s)(y-t).
```

Expanding,

```text
(y^4-s)(y-t) = -s y + st - t y^4 + y^5.
```

The high coefficients are `c=-t` and `d=1`, so

```text
d/c = -t^{-1}.
```

This proves the lemma.

## Translation back to the weighted half-turn branch

For a half-turn quotient set `S`, let

```text
T = pi(S) subset mu_8
```

be the projection to quotient fibers.  The classifier in Note 0346 found
exactly the `(4,1)` projection supports above, with the fiber-weight profile

```text
0^3, 1^2, 2^3.
```

The two singleton fiber signs do not affect the five-root lemma.  They only
choose which lift of each quotient point is present in `S`.  This explains
the observed count:

```text
8 choices of singleton quotient root t
* 4 choices for which point of the full coset has weight 1
* 4 choices of two singleton signs
= 128 weighted sets.
```

For both even and odd half-turn eigenrows, the same quotient polynomial
`f_{s,t}` controls the high direction.  This is why the classifier outputs
matching even/odd ratios for all three primes.
